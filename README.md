#  🗑️ Waste_Classification_CNN_Model
"Every day, tons of recyclable waste end up in landfills simply because it isn't sorted correctly. What if a computer could identify garbage like paper, plastic, glass, or metal in just a fraction of a second? That's exactly the problem my CNN-based Waste Classification system aims to solve."

A Deep Learning project that classifies waste images into six categories using a Convolutional Neural Network (CNN). The model is trained on the TrashNet dataset to automate waste segregation and support efficient recycling.

## 📌 Features
- Image classification using CNN
- Six waste categories:
  - Cardboard
  - Glass
  - Metal
  - Paper
  - Plastic
  - Trash
- Built with TensorFlow and Keras
- Model evaluation and prediction on unseen images

DATASET LINK : https://www.kaggle.com/datasets/feyzazkefe/trashnet

## 🛠️ Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## 📊 Results
- Validation Accuracy: **71.56%**
- Validation Loss: **0.91**
